# GitHub Jobs Aggregator

Aggregates remote job postings from GitHub Issues across multiple repos. Browse, filter, and apply to jobs with AI-generated personalized cover letters.

## Features

- Fetches open issues from a list of GitHub job-board repos (e.g. `backend-br/vagas`, `frontendbr/vagas`)
- Filters by configurable keywords against issue title + body
- Groups results by date (newest-first)
- Minimal dark-themed web UI at `http://localhost:3000`
- REST API endpoints for jobs, repos, profile, and apply
- Daily cron job at 8am to auto-refresh listings
- Repo discovery via Claude CLI + WebSearch
- **Apply to jobs** with a personalized cover letter generated by Claude using your CV

## Prerequisites

- **Node.js 22+**
- **GitHub personal access token** (with `public_repo` scope) for API rate limits
- **Claude CLI with an active subscription** â€” required for repo discovery and cover letter generation.
  Alternatively, Claude CLI can be configured with a local model setup (see [Claude Code docs](https://docs.claude.com/claude-code)).

## Setup

```bash
npm install
cp .env.example .env
# Edit .env and add your GITHUB_TOKEN
npm run dev
```

Open `http://localhost:3000`.

## Environment variables

| Variable | Description |
|---|---|
| `GITHUB_TOKEN` | GitHub personal access token (required) |
| `KEYWORDS` | Comma-separated keywords to filter issues (e.g. `remote,developer,react`) |
| `DAYS_BACK` | Lookback window in days (default: 14) |
| `PORT` | Server port (default: 3000) |

## API

| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/jobs` | Full date-grouped jobs object |
| `GET` | `/jobs/:date` | Jobs for a specific date (`YYYY-MM-DD`) |
| `POST` | `/jobs/trigger` | Run aggregation now |
| `GET` | `/repos` | List tracked repos |
| `POST` | `/repos/discover` | Find new repos via Claude + WebSearch |
| `GET` | `/profile` | Get your profile (404 if none) |
| `POST` | `/profile` | Create/update profile (multipart: `name`, `email`, `cv` file) |
| `DELETE` | `/profile` | Delete profile and CV |
| `POST` | `/apply` | Generate a cover letter for a job (body: `{ jobUrl }`) |

## Applying to jobs

1. Go to the **Profile** tab
2. Enter your name, email, and upload your CV as a PDF
3. Back on the **Jobs** tab, click **Apply** on any job card
4. Claude fetches the full job description, reads your CV, and generates a personalized cover letter
5. Review/edit the generated letter
6. If a contact email is found in the posting, click **Open Email** (mailto) â€” otherwise use **Copy to Clipboard**

## Privacy

All user data â€” profile, CV, jobs cache â€” lives in the local `data/` directory which is gitignored. No data is sent to third-party services except:
- The Claude CLI (processes locally on your machine)
- The GitHub API (for fetching job issues, using your token)
